# MyProject

## 概要
三人称視点でダンジョンを探索し、敵と戦いながら最奥のボス撃破を目指すターン制コマンドRPGです。
「シンプルな構成と、短時間で完結するゲーム体験」をコンセプトに制作しました。

## ゲーム内容

### ■フィールド
- 3D探索：三人称視点でのダンジョン探索(通常エリア3、ボスエリア1)
- シンボルエンカウント：敵シンボルとの接触で戦闘へ遷移
- リソース管理：レベルアップ時HP全回復

### ■戦闘システム
- 完全ターン制(プレイヤー → 敵)
- コマンド：攻撃(Attack) / 防御(Guard) / 逃走(Escape)
- ボスはHPが半分以下で強力なため攻撃を使用

### ■成長要素
- 最大レベル：5(必要経験値：現在レベル * 15)
- 経験値：雑魚敵15 / ボス60

## 操作方法
- 移動：WASD  
- カメラ操作：マウス  
- 決定 / 攻撃：クリック

## 使用技術
- **C++**  
  - 戦闘ロジック（ダメージ計算、ターン管理、リソース管理）  
  - キャラクター基底クラスの設計  

- **Blueprint**  
  - 入力制御、カメラワーク、VFX/SE制御  
  - 戦闘遷移（開始・終了）の演出  

- **UMG**  
  - HPバー、コマンドUI、リザルト画面の実装
 
## 工夫した点  
- **C++とBlueprintの役割分担**  
  ロジックはC++、演出や調整はBlueprintで実装することで、可読性と開発効率を両立しました。  

- **GameInstanceによるデータ保持**  
  レベル遷移時もHPや経験値が保持されるようにし、状態管理を一元化しました。

## 苦労した点
- **戦闘後のHPや経験値の保持処理**
  戦闘と探索でレベルが分かれているため、戦闘終了後にプレイヤーのHPや経験値が初期化され正しく引き継がれない問題がありました。

  → GameInstanceでプレイヤーの状態を管理し、戦闘前後でデータを保持・復元する仕組みを実装することで解決しました。
